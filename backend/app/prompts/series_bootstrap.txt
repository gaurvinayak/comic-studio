- Keep it original; do not copy named characters from existing franchises.

Return ONLY a single raw JSON object (no markdown fences, no commentary) with EXACTLY this shape:

{
  "title": "<short, evocative series title>",
  "premise": "<2-3 sentence hook describing the central conflict and what makes it compelling>",
  "genre": "<primary genre, e.g. sci-fi noir, cozy fantasy, superhero satire>",
  "tone": "<emotional register, e.g. wry and fast-paced; dark and brooding>",
  "art_style": "<one concrete visual style for the whole series: line work, coloring, lighting, mood>",
  "world_bible": "<2-4 sentences establishing the setting, the rules of this world, and the status quo>",
  "characters": [
    {
      "name": "<distinct character name>",
      "role": "<protagonist | antagonist | supporting>",
      "concept": "<one-line concept / archetype>",
      "personality": {
        "traits": ["<trait>", "<trait>", "<trait>"],
        "motivations": "<what drives them>",
        "voice": "<how they speak: cadence, vocabulary, verbal tics>"
      },
      "appearance": "<concrete, fixed visual description: age range, body build, face shape, hair (color/style), skin tone, eye color, signature outfit, and 2-3 identifying colors>"
    }
  ]
}

Provide 3 to 5 characters with at least one protagonist and one source of opposition.
